rustfmt

Formatter for Rust.

You can install rustfmt with rustup like so:

rustup component add rustfmt

and then run it like this:

cargo fmt

Github Actions

To run rustfmt in Github Actions:

  lint:
    runs-on: ubuntu-latest
    steps:

    - uses: actions/checkout@v3

    - name: Format
      run: cargo fmt --all --check
