# Tools

Inspired by <https://jyn.dev/tools/>.

## Basics

I use these every day.

- [bat](https://github.com/sharkdp/bat): `cat` replacement with highlighting
- [delta](https://github.com/dandavison/delta): `diff` replacement
- [entr](http://eradman.com/entrproject/): watch files
- [fd](https://github.com/sharkdp/fd): `find` replacement
- [fzf](https://github.com/junegunn/fzf): responsive UIs for everything
- [gh](https://cli.github.com/): GitHub CLI
- [ghcid](https://github.com/ndmitchell/ghcid): Haskell dev tool
- [glab](https://docs.gitlab.com/cli/): GitLab CLI
- [helix]: editor, used with [custom keybindings]
- [jq](https://jqlang.org/): JSON processing
- [kmonad](https://github.com/kmonad/kmonad): keyboard manager
- [rg](https://github.com/BurntSushi/ripgrep): `grep` replacement
- [taskwarrior](https://taskwarrior.org)
- [tealdeer](https://tealdeer-rs.github.io/tealdeer/): `man`, but brief
- [zoxide](https://github.com/ajeetdsouza/zoxide)
- [zsh](https://www.zsh.org/): shell, see below

[helix]: https://github.com/helix-editor/helix
[custom keybindings]: https://github.com/langston-barrett/spacelix

## Custom

Little utilities I wrote.

- [`copy` and `paste`]: copy and past from the shell, on Linux or macOS
- [`notify`]: send a notification on Linux or macOS
- [`pick`]: pick something at the terminal (just some `fzf` flags)

[`copy` and `paste`]: https://github.com/langston-barrett/dots/blob/576dc9072e68c5ae33d568f29a4d2c9acedecec7/files/sh.d/copy-paste.sh
[`notify`]: https://github.com/langston-barrett/dots/blob/576dc9072e68c5ae33d568f29a4d2c9acedecec7/files/scripts/bin/notify
[`pick`]: https://github.com/langston-barrett/dots/blob/576dc9072e68c5ae33d568f29a4d2c9acedecec7/files/scripts/bin/pick
